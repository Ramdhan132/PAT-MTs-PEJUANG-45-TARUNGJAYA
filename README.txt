Ujian Online - Demo (Vite + React)

Cara menjalankan (lokal):
1. Pastikan Node.js terpasang (versi 16+).
2. Extract zip dan buka folder proyek.
3. Jalankan:
   npm install
   npm run dev
4. Buka http://localhost:5173

Catatan:
- Guru dapat mengunggah file .docx berformat sederhana (lihat contoh di UI).
- Untuk produksi: gunakan backend untuk parsing, penyimpanan, dan autentikasi.
